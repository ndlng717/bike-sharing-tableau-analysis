# Bike Sharing System Analysis (Tableau)

## Project Overview
This project analyzes bike-sharing trip data to uncover patterns in station demand, rider demographics, and trip behavior. The analysis highlights key insights about user types, trip duration patterns, and station utilization.

## Key Questions
- Which stations generate the highest trip demand?
- How do subscribers and customers differ in usage?
- What are the most common trip durations?
- Which rider demographics dominate usage?

## Key Insights
- Trip demand is concentrated in a limited number of stations.
- Subscribers account for the majority of trips (87%).
- Most trips last between 5 and 10 minutes.
- Riders aged 35–40 generate the largest share of trips.
- Subscriber trips decreased by 25% between March and May.

## Dashboard Features
The Tableau dashboards include:

- KPI overview
- Station demand analysis
- User demographic breakdown
- Trip duration segmentation
- Monthly usage trends
- Key insights and recommendations
